# My App

This project is built with [Vite](https://vitejs.dev/) and React.

## Installation

Install dependencies with:

```bash
npm install
```

## Development server

Start the development server with:

```bash
npm run dev
```

## Build

Create an optimized production build with:

```bash
npm run build
```

## Lint

Run ESLint to analyze and fix issues:

```bash
npm run lint
```
